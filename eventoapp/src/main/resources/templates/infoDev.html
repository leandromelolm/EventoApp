Foram feitos incrementos de funcionalidades e outros ajustes.

  • settingsDetalhes
    • Esta aplicação web permite que o usuário cadastre eventos e deixe-os Aberto para o cadastro de participantes.
    • Para cadastrar evento o usuario precisa se cadastrar e logar no sistema.
    • É permitido também ao usuário que estiver logado no sistema editar e excluir seus proprios eventos.
    • Os eventos com status Aberto é permitido que qualquer um faça o cadastro de participantes.
    • Caso o evento esteja com o status Pausado ou Encerrado o acesso para ver detalhes do evento fica restrito ao usuario que realizou o cadastro do evento.
    • Com o status Finalizado ninguém tem permissão para abrir detalhes do evento.
    • O usuário com perfil Administrador pode excluir os eventos.
    • No frontend do projeto é usado recursos dos Frameworks Materialize e Boostrap.
