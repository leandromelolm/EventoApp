Projeto EventoApp

Projeto desenvolvido como aprendizagem.

Feito com base no Curso Spring Boot: Criando uma aplicação Java Web com a professora Michelli Brito.

Foram feitos incrementos de funcionalidades e outros ajustes.

Mas a base para construção foram as aulas no youtube e o projeto no github da professora Michelli.

Aqui meu Projeto no github.
  • new_releasesNovidades
    • Página Home (/home) usa bootstrap 5.2 nos cards.
    • Página Index (/) usa Materialize.
    • Menu com uso do Materialize.
  • settingsDetalhes
    • O App permite que o usuario cadastre, edite e exclua seus eventos.
    • O evento permite que qualquer pessoa faça cadastro
    • Caso o evento esteja com o status Pausado ou Encerrado o acesso para ver detalhes do evento fica restrito ao usuario que realizou seu cadastro.
    • O usuario Administrador pode excluir os eventos.
  • buildImplementacões Futuras

    • Remover o Materialize e adicionar o Bootstrap em toda aplicação.
    • Criar campo para usuario pesquisar evento.
    • Adicionar paginação na lista de eventos.
    • Adicionar paginação na lista participantes cadastrados nos eventos.
    • Entre outros ajustes.
